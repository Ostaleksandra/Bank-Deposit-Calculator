# Расчёт дохода по вкладу

Учебный проект на Python.

## Описание

Программа рассчитывает доход по банковскому вкладу в разных банках на основе введённой суммы и процентной ставки.

## Как работает

Пользователь вводит сумму вклада, после чего программа:

* вычисляет прибыль для каждого банка
* выводит список доходов
* показывает максимальную возможную прибыль.

## Цель проекта

Практика работы со словарями, списками и базовыми вычислениями в Python.
